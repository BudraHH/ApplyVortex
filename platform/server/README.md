# ApplyVortex Server

Backend API for ApplyVortex, hosted on Hugging Face Spaces.

## Overview
This is a FastAPI-based backend that orchestrates job applications using local agents.

- **Stack**: FastAPI, Postgres (Neon), Redis (Internal), Celery.
- **Port**: 7860
